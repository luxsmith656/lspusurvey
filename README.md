# LSPU-LB Pulsong Ka-PiYu

Survey portal for the 5th Gawad Parangal 2026 event.

## Features

- Fixed event QR route: `/gawad-parangal-2026`
- Production QR link: `https://lspusurvey.vercel.app/gawad-parangal-2026`
- Pre-event first-impression 5-star rating with optional comment
- Admin-controlled Organization M&E form gate
- Separate pre-event and Organization M&E results with CSV exports
- Offline-safe response queue that syncs when the device reconnects

## Development

```sh
npm install
npm run dev
```
